# artifacts/ — source material (inputs)

Drop the novel's existing material here: drafts, fragments, outlines, character
sketches, worldbuilding notes, timelines, anything. Any format is fine (Markdown
preferred; `.txt`/`.docx`/`.pdf` ok).

These files are **read-only inputs** — the agents synthesize from them but never edit
them. New writing goes in `../manuscript/` and `../docs/`.
